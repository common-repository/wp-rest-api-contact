== Description ==

This is the WP REST API NPA plugin that allows sending data from the client to the server via the API path. 

If you want to contact me for any reason, feel free to email me at chithien175@gmail.com or contact me on facebook: https://www.facebook.com/thien.pham.5074

== Installation ==
1. Upload the `wp-rest-api-contact` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

- Create to the Rest API Contact endpoint.
- Show the contact list.
- On/Off receive an email when a new contact
- Create to the Rest API Subscribe Newsletter endpoint.
- Show the list all subscribers.
- Send mail to list all subscribers.
- Open / close modules of the WP REST API NPA plugin.
